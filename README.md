# CareerCompass AI

An AI-powered student career guidance platform that helps students analyze their skills, assess internship readiness, generate learning roadmaps, and discover suitable projects.

## Features

### Career Analysis
- Analyze student skills
- Identify skill gaps
- Suggest career paths

### Internship Score
- Evaluate internship readiness
- Provide improvement suggestions

### Learning Roadmap
- Generate personalized learning plans
- Track progress toward career goals

### Resume Analyzer
- Review resume details
- Highlight strengths and weaknesses

### Project Recommender
- Suggest projects based on skills and interests
- Improve portfolio development

## Tech Stack

- Python
- Streamlit
- Pandas
- Plotly
- Git
- GitHub

## Installation

Clone the repository:

```bash
git clone https://github.com/Jinesh1001/AI-Student-Success-Agent.git
```

Go to the project folder:

```bash
cd AI-Student-Success-Agent
```

Install dependencies:

```bash
pip install -r requirements.txt
```

Run the application:

```bash
streamlit run app.py
```
