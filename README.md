# 🎓 AI Student Success Agent
### Your Personal Career & Internship Mentor

## Overview

StudentPath AI is an AI-powered student career development platform that helps students:

* Discover career paths
* Identify skill gaps
* Analyze resumes
* Track internship readiness
* Generate learning roadmaps
* Practice interview questions
* Get personalized career guidance

---

## Features

### 📊 Dashboard

* Student profile management
* Career readiness tracking
* Internship readiness tracking
* Personalized action plan

### 🎯 Career Analysis

* Career recommendations
* Skill gap analysis
* Learning roadmap generation

### 📄 Resume Analyzer

* PDF Resume Upload
* Resume scoring
* Skill detection
* AI career advice

### 💼 Internship Finder

* Internship role recommendations
* Skill improvement suggestions

### 🎤 Interview Coach

* Technical interview questions
* Career-specific preparation

### 🤖 Career Assistant

* AI-style career guidance chatbot

### 🚀 Project Recommender

* Domain-based project suggestions
